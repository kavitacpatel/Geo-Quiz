# Geo-Quiz
Geo-Quiz Game
Fun App for iOS. It has 3 flag buttons. User is able to hear question and choose correct answer by pressing button.
After got correct answer it take you to next quiz.

---------------------------------------------------------------------------------------------------------------------------
This iOS App is Universal.
